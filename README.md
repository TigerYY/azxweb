## 使用说明

1. 直接在浏览器中打开 `index.html` 即可访问网站
2. 所有页面通过导航栏链接相互连接
3. 移动端会自动显示响应式布局

## 自定义配置

### 颜色主题
在 `css/style.css` 文件的 `:root` 变量中修改：
```css
:root {
    --primary-color: #084861;      /* 主色调 */
    --secondary-color: #0a1936;    /* 次要色调 */
    --accent-color: #00d4ff;       /* 强调色 */
}
```

### 内容修改
直接编辑对应的 HTML 文件即可修改页面内容。

## 注意事项

1. 图片资源：目前使用占位符，需要替换为实际产品图片
2. 地图功能：联系页面中的地图需要集成实际的地图服务（如百度地图、高德地图）
3. 表单提交：联系表单需要配置后端服务来处理提交
4. Favicon：需要添加实际的网站图标文件
